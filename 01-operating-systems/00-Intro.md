# LEVEL 1 — Operating Systems (The Heart of Everything)

## Big Picture (Intuition First)

An **Operating System (OS)** is not magic.

Think of it as:

> **A very strict manager** that controls CPU time, memory, files, and hardware — so multiple programs can run *safely* at the same time.

Every modern system problem eventually comes down to:

- CPU being busy
- Memory running out
- Disk being slow
- Too many things happening at once

Level 1 teaches you **why systems behave the way they do**.
