# Tech Foundation

This repository documents my understanding of core computer systems concepts,
built step-by-step with a focus on **mental models**, not memorization.

The goal is to understand how a computer goes from
**running a program → communicating over a network → serving real users**.

This is a personal learning repository, but structured clearly enough
to explain my thinking to others.

---

## Learning Philosophy

- Concepts first, tools second
- One topic at a time
- Focus on *why things exist* and *how they connect*
- Written in my own words after studying

These notes are meant to be **revisited and refined**, not treated as final.

---

## Learning Order

1. How computers work (foundations)
2. Operating Systems
3. Networking
4. Linux Internals
5. Containers & Isolation
6. Performance & Reliability

Each folder builds on the previous one.
